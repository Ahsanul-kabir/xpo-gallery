How does javascript works ?

JavaScript is scripting languages that likes client-side scripting languages.It means that a language like JavaScript runs inside browser. Now, I would like to describe how JavaScript basically works. It works like a cycle. Firstly I can say, The web browser loads a web page, then parses the HTML file, and creates DOM from the contents. The DOM show a live view of the web pages to our JavaScript code. Secondly, The browser merge or grad everything linked to the HTML file, like images and CSS files. The CSS codes comes from the CSS parser. Then create a web page. Thirdly, Then, the browsers' JavaScript engine likes Google Crome v8 engine loads JavaScript files but it does not run the code immediately. It waits for the HTML and CSS to finish loading all files. Then if anything is updated DOM will be re-render. This likes work a cycle.

Write the differences between local storage and session storage ?

Local storage:
1. Local Storage is basically use 5MB or 10MB.
2. Basically user can read local storage.
3. Local storage is like a permanent storage of a web app.
4. Local storage data is not transferred to server.

Session storage:
1. Session storage is basically use 5MB.
2.Basically user can not read Session storage.
3.Local storage is like a some period storage of a web app.
4.Session storage data is not transferred to server.
